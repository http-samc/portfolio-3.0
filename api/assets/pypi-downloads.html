This API allows you to get the total amount of downloads any PyPi project has gotten throughout its existence. It employs the use of PyPi stats and Badge Maker. The endpoints are listed below:

  • /:packageName allows you to get the total number of downloads and the total amount of days a package has been available. The JSON returned looks like:
    {
         "downloads": <(int) number of downloads>,
         "days": <(int) number of days available>
    }
    See an example here.

  • /badge/:packageName returns a svg representing the total number of downloads that can be used in README (or similar) files. The badge name is 'PyPi Downloads' (customizable) and the message is the exact number of downloads (with commas). To customize your badge, you can pass the following query parameters in:
    • labelColor (body color of the left-side of the badge, defaults to #555, do NOT pass in the '#' when calling, only the hex integer)
    • messageColor (body color of right-side of the badge, defaults to #4c1, do NOT pass in the '#' when calling, only the hex integer)
    • style (style of the badge, defaults to flat, options: plastic, flat, flat-square, for-the-badge, social)
    • label (label of the badge, defaults to PyPi Downloads')
      • See a fully customized example here, or a basic one here.
