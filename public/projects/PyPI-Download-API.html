PyPI Download API

The simplest badge api for cumulative PyPI downloads!

This API allows you to get the total amount of downloads any PyPI project has gotten throughout its existence. It employs the use of PyPI Stats (which provides monthly download breakdowns) and Badge Maker (which powers the badge endpoint). I primarily made this API for myself, as I like to know how many people are using packages I publish so I know what I need to work on. The endpoints (which use https://www.smrth.dev as their base) are listed below:

JSON API

  • /api/pypi-downloads/:packageName: Get the number of downloads and number of active days of a PyPI package.
  • Sample 'GET' URL
  • Return Schema:
    {
  "downloads": <(int) number of downloads>,
  "days": <(int) number of active days>
}
  • Sample Return:
    {
  "downloads": 19454,
  "days": 230
}

README Badge API

  • /api/pypi-downloads/badge/:packageName: Get a README-file ready badge that contains your project's total PyPI downloads.
